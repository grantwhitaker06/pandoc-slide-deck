How to use

  1. Download Pandoc
  2. Fork this repo: pandoc slidedeck creator
  3. Use pandoc markdown and create your slides
  4. Never use powerpoint again

Notes on usage

  • Pandoc key/val pairs are passed into the coorisponding $<var name>$
  • Variables are defined using link labels at the top of the Markdown file
  • They will not show up in the document
  • I realize this is hacky, but necessary to keep Markdown integrety
